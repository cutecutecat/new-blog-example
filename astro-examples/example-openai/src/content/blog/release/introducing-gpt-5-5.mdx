We're releasing GPT-5.5, our smartest and most intuitive-to-use model yet, and the next step toward a new way of getting work done on a computer.

GPT-5.5 understands intent faster and can carry more of the work itself. It excels at coding, researching online, analyzing data, creating documents and spreadsheets, operating software, and moving across tools until a task is complete.

#### Capability summary

- **Agentic coding:** Long-horizon planning, repo-level edits, debugging, and verification
- **Computer use:** More stable execution in multi-step UI workflows
- **Knowledge work:** Better synthesis for docs, slides, and spreadsheet-heavy tasks
- **Scientific research:** Stronger iterative reasoning and evidence refinement

## Model capabilities

The gains are especially strong in agentic coding, computer use, knowledge work, and early scientific research. GPT-5.5 delivers this increase in intelligence without sacrificing practical latency.

## Availability and pricing

At launch, GPT-5.5 rolled out first in ChatGPT and Codex tiers, then expanded to API access with additional safeguards.

| Surface | Availability | Notes |
| --- | --- | --- |

| Codex | Paid tiers | Strong gains on implementation and long-running tasks |
| API | Available with additional safeguards | See release updates and system card |
